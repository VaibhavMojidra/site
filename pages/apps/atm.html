Type : Software

IDE/Tools: Netbeans IDE, Microsoft SQL Server 2012
Programming Language/Framework : Java, Java Swing, Java AWT

Concepts : JDBC, SQL, Microsoft SQL Server 2012, Dimension, ActionListener, ItemListener, KeyListener, MouseListener, IOException, Exception Handling, JButton, BorderFactory, AudioInputStream, JFrame, JOptionPane, JPasswordField, SwingConstants and more...

Description : ATM is Windows desktop application developed using Java Swing. It is a demonstraion of ATM + CDM in one software

• User can login/logout in machine.
• User can deposit their money.
• User can withdraw their money.
• User can transfer one account to another.
• User can check balance in their account.
• User can change pin/reset pin.
