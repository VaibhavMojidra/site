Programming MCQ App

Programming MCQ App

Type : Software
Platform : Android
IDE/Tools : Android Studio
Programming Language/Framework : Kotlin
Database: Firebase - Realtime Database, FirebaseMessaging

Concepts : In-App-Review Google Play API, In-App-Update Google Play API, RecyclerView, Custom Alert, itextPDF, FirebaseMessaging, Firebase Realtime Database, JSON, NoSQL, Kotlin, SharedPreferences, Glide, GooglePlayServices, TextInputLayout, Dark Mode, Animation and more...

Description : Programming MCQ App is an android Application made using Kotlin Programming Language to improve as well as test programming knowledge on various languages.

✓ It has a huge set of questions from which random 30 Questions are set for one test and shuffle options.
✓ It allows you to get result in the form of PDF in the end of each test which contains all the questions and answers (in the test) and their result (correct and incorrect).
✓ It has dark mode for less eye strain.
✓ It allows you to book mark any question while giving test and has a exam summary to view before submitting.
✓ Languages can be dynamically updated/added from server.
✓ It uses firebase real time database for instant result and fast server.
