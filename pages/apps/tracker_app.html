Tracker App

Type : Software
Platform : Android, IOS
IDE/Tools : Microsoft Visual Studio Code (VS Code)
Programming Language/Framework : Javascript es6, React Native
Database tool : ngrok
Database: MongoDB

Concepts : Components, State Management, useState, useEffect, ngrok, useReducer,MongoDB, Hashing using Bcrypt, Rainbow Table Attack - Salting, React Navigation v4, React maps, Authentication, JWT, Nodemon, AsyscStorage

Description : Tracker App is an tracking application which allow you to record and save the tracks/routes that you have been walk through. This app is made over React Native hence it runs on both IOS and Android. And all the data is stored on MongoDB Cloud.
