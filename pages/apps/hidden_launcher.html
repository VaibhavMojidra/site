Hidden Launcher

Type : Software
Platform : Android
IDE/Tools : Android Studio
Programming Language/Framework : Java

Concepts : In-App-Review Google Play API, BaseAdapter, LinearLayout, BottomSheetBehavior, SharedPreferences, BroadcastReceiver, RecyclerView, WallpaperManager, RelativeLayout, SwitchCompat, Custom Alert and more...

Description : Hidden Launcher is an android Application made using Java Programming Language. It is a Launcher App with following features:

• It enables you to hide your bottom app drawer from Main/Home screen.
• It enables you to customize app name color inside your app drawer.
• It enables you to customize app drawer color.
• It enables you to customize drawer app transparency.
• It enables you to customize clock.
• It enables you to customize clock color.
• It enables you to hide apps from its settings.
• It enables you to hide status bar.
• It enables you to set grid size of your apps in drawer.
• It enables you to set, preview and change wallpaper.
